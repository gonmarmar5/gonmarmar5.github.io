GridExplorers

This project is a dynamic simulator that models agent movement on a grid. Agents aim to reach a central square, adapting to changes using A* algorithms and density-based adjustments. The program allows for random agent relocation and target square repositioning. Visualization features explore agent interactions with changing goals and study emergent dynamics.
